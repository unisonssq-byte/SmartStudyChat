Создай проект Telegram-бота **Custos | Чат-менеджер** — упрощенный вариант Iris, на Python с использованием библиотеки aiogram. Проект должен включать:

### 1. Основное меню бота

* В самом боте есть только команды:

  * `/start` — запуск бота, отображает **главное меню** с обложкой (сгенерировать изображение) и описанием функций бота.
  * `/help` — выводит сгенерированное изображение с названием "Команды" и краткое описание функций.
* Все остальные команды работают **только в чатах**, не доступны из личного чата с ботом.
* Инлайн кнопка на главном меню: **"Добавить в чат"**.
* Ниже главного блока: обычные кнопки:

  * **"Мои чаты"** — показывает список чатов, где добавлен бот, с сгенерированным изображением сверху. Названия чатов кликабельны (гиперссылка на чат).
  * **"Команды"** — отображает картинку с заголовком "Команды" и описание: *"Полный список комманд в нашей статье"* (слово "статья" — гиперссылка на [https://teletype.in/@unisonqq/custoscommands](https://teletype.in/@unisonqq/custoscommands)).

### 2. Цветовая палитра изображений

* Темный стиль: фиолетовые, синие, черные и белые цвета.
* Фон всегда темный, текст — светлый.

### 3. Генерация изображений

* Каждое меню, блок или команда должна иметь **сгенерированное изображение с названием блока или команды**.
* В корень проекта положить **аватарку самого бота**, соответствующую цветовой палитре.

### 4. Команды и функции бота (только для чатов)

#### 4.1 Команды повышения (/upstaff)

* Формат: `/upstaff [число] [username/ссылка/в ответ]`.
* Повышает ранг пользователя на указанное количество единиц.
* Ранги: участник → модератор → администратор → владелец.
* Проверка прав:

  * Участник и модератор — команда недоступна ("Слишком низкий ранг").
  * Администратор — может повышать модераторов и участников, владельцев нельзя.
  * Владелец — может повышать любого; если повышение до владельца — появляется **инлайн кнопка "Подтвердить"**, только владелец может нажать, иначе уведомление сверху: "Эта кнопка не для вас ^-^".
* Сообщения после повышения: `[username] повышен в ранге, теперь он [ранг]`.

#### 4.2 /staff

* Выводит список участников по рангам: Владелец, Администраторы, Модераторы.
* Доступна всем.

#### 4.3 Команды обычных пользователей

1. **/ban \[username/ссылка/в ответ] причина** — исключает пользователя из чата (черный список). Только администратор/владелец.
2. **/warn \[username/ссылка/в ответ] причина** — выдает варн; при 5 варнах — бан. Доступно модератору (1 раз в час), администратору и владельцу.
3. **/kick \[username/ссылка/в ответ] причина** — исключение пользователя на время, не в черный список. Модератор 1 раз в 15 минут, администратор/владелец без ограничений.
4. **/me (или кто я)** — отображает: имя пользователя (внутреннее имя в боте, если установлено), ссылка на пользователя, количество сообщений, ранг, описание, сгенерированное изображение "Описание чатера".
5. **/you \[username/ссылка/в ответ] (или кто ты)** — информация о другом пользователе, аналогично /me.
6. **/nickname (+ник или +имя)** — устанавливает внутреннее имя для чата.
7. **/description (+опис или +описание)** — устанавливает описание пользователя.
